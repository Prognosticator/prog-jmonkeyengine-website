jMonkeyStore is Now Open!

After years of requests and attempts, a software store (a.k.a an asset store) for jmonkey has finally been created and is now open to registration. Everybody can register and submit their software! https://store.jmonkeyengine.org/ Currently only free open-source software can be submitted.
